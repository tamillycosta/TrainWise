# TrainWise
TrainWise é um aplicativo de gerenciamento de treinos focado em treino consciente, acompanhamento de progressão e análise de desempenho, com suporte completo a uso offline. Desenvolvido com .NET MAUI e ASP.NET, possui arquitetura preparada para sincronização e futuras funcionalidades de análise avançada.

## Tech Stack
- .NET MAUI
- SQLite
- ASP.NET Web API (future sync)
